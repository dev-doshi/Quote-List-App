# Quote-List-App

dependencies:
Xcode 12.4

1. Open Xcode

2. Click on "Clone an existing project"
   Image: https://bit.ly/3tPgDk9
   
3. Copy/Paste "https://github.com/dev-doshi/Quote-List-App" into the Search Bar 

4. Press "Clone"

5. Select where you want to save your project and then press "Clone" again

6. Press ⌘ + R to run the project or press ▷ symbol in the top-left corner
